1 ) Difference Between var, let, and const --- >
1.1 - var ---
.1) Old way to declare variables in JavaScript
.2 ) Can be redeclared
.3 ) Can be updated
.4 ) Function scope
1.2 - let ---
ES6 - modern JavaScript
.1 ) Cannot be redeclared
.2 ) Can be updated
.3 ) Block scope
1.3 - const ---
Used for constant variables
.1 ) Cannot be redeclared
.2 ) Cannot be updated
.3 ) Block scope

2 ) Spread Operator (...) --- >
The spread operator (...) is used to expand or copy elements of an array or object.
code :
const numbers = [1,2,3];
const newNumbers = [...numbers,4,5];
console.log(newNumbers); // [1,2,3,4,5]

3 ) Difference Between map(), filter(), and forEach() --- >
forEach() ---
1.Used to loop through an array
2.Does not return a new array
map() ---
1.Used to transform data
2.Returns a new array
filter()---
1.Used to select elements based on a condition
2.Returns a new array

4) Arrow Function --- >
An arrow function is a shorter way to write functions in JavaScript.
syntax -
const data = (parameter) => (one line condition) // if { multiline condition }

5) Template Literals --- >
Template literals allow you to insert variables inside strings easily
use backticks ${} instead of quotes.
exm :
const name = "SaKib";
const age = 22;

console.log(`My name is ${name} and my age is ${age}`); // My name is SaKib and my age is 22
